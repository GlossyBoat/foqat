### 数据统计 statdf 
----------
* #### 描述

统计数据的情况。包括：均值、方差、最小值、最大值、分位数值（25%、50%、75%）。输出数据框列为统计参数、行为变量。

* #### 用法
``` r
statdf(x, n = 2)
```
* #### 参数

| 变量名             | 含义                              | 默认值                     |示例值                                 |
| ------------------| ----------------------------------|----------------------------|--------------------------------------|
| `x`               | 待统计的数据框                     |                            |                                      |
| `n`               | 结果数据保留的小数位数              | 2                          |                                      |

* #### 示例
